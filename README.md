# VLSI Design Internship – Task 2
## Introduction to Verilog HDL and RTL Design of Basic Combinational Circuits

This repository contains the implementation of **Task 2** from the **MainCrafts Technology VLSI Design Internship**. The objective of this task is to learn the fundamentals of **Verilog HDL**, understand **RTL (Register Transfer Level) Design**, and implement basic combinational logic circuits using Verilog.

---

## 📌 Objectives

- Learn Verilog HDL fundamentals
- Understand RTL design concepts
- Design basic combinational circuits
- Write synthesizable Verilog code
- Create testbenches for verification
- Simulate and analyze waveform outputs

---

## 🛠️ Tools Used

- Xilinx Vivado 2025.2
- Verilog HDL
- Vivado Simulator (XSim)
- Windows 11

## 🧪 Verification

The following testbenches were developed:

- Logic Gates Testbench
- Half Adder Testbench
- Full Adder Testbench

Each module was simulated using **Vivado Simulator**, and the outputs were verified against the corresponding truth tables.
